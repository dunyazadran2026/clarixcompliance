# Clarix Compliance Consulting

Website for [clarixcompliance.com](https://clarixcompliance.com)

Independent regulatory consultancy for MedTech and Pharma — MDR/IVDR, ISO 13485, EU AI Act, GxP.

## Files
- `index.html` — Main website (all pages as sections, DA/EN language toggle)
- `styles.css` — Brand styles (Deep Navy #163A5F, Teal #1C8C8C)
- `script.js` — Navigation, language switching, brand SVGs
- `CNAME` — Custom domain mapping
- `404.html` — Custom 404 page
- `favicon.ico` — Browser icon (placeholder)

## Deploy
GitHub Pages → Settings → Pages → Source: main / root
